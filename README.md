# Today Only To-Do App

A simple to-do app created for the MCPs and Skills Across the Software Lifecycle assignment.

## Features
- Add tasks
- Mark tasks as complete
- Delete tasks
- Save tasks with localStorage

## Files
- index.html
- style.css
- app.js
- docs/prd.md

## How to Run
Open `index.html` with Live Server in VS Code.
